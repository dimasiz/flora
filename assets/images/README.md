# Изображения

В этой папке можно хранить дополнительные изображения для приложения.

На данный момент приложение использует эмодзи для отображения животных и растений,
что обеспечивает:
- Быструю загрузку
- Адаптивность
- Яркий мультяшный стиль

При необходимости можно добавить:
- Фоновые изображения
- Иллюстрации для карты
- Аватары персонажей
