# Звуки для игр

В этой папке должны находиться звуковые файлы:

- `correct.mp3` - звук правильного ответа
- `wrong.mp3` - звук неправильного ответа  
- `win.mp3` - звук победы/завершения уровня
- `click.mp3` - звук клика

## Где взять звуки?

Бесплатные звуки можно скачать с:
- https://freesound.org/
- https://mixkit.co/free-sound-effects/
- https://www.zapsplat.com/

Рекомендуемые типы звуков:
- Для `correct.mp3` - короткий позитивный звук (ding, chime)
- Для `wrong.mp3` - короткий звук ошибки (buzz, honk)
- Для `win.mp3` - победная мелодия, аплодисменты
- Для `click.mp3` - мягкий звук клика
